# User Dashboard

A responsive React + TypeScript application that displays users fetched from an API. The project includes searching, sorting, loading states, and reusable UI components.

---

## Features

- Fetch users from API
- Responsive user cards
- Search users by name
- Sort users
- Loading state
- Error handling
- Redux Toolkit state management
- TypeScript support

---

## Tech Stack

- React
- TypeScript
- Redux Toolkit
- React Redux
- Vite
- Tailwind CSS (or CSS Modules if applicable)
- Axios

---

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/user-dashboard.git
```

Go to the project

```bash
cd user-dashboard
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
